Video-Ordner für die aktuellen Leistungen.

Struktur:
- `/videos/leistungen/decken-abhaengen`
- `/videos/leistungen/waende-stellen`
- `/videos/leistungen/estrich-boden`
- `/videos/leistungen/dachschraegen`
- `/videos/leistungen/sonstiges`

Je Leistung:
- ein Ordner
- ein Video für alle Geräte
- optional zusätzlich eine `.webm`-Variante

Empfohlene Dateinamen:
- `scene.mp4`
- optional `scene.webm`

Hinweise:
- Die Datei `BENOETIGTES_VIDEO.txt` im jeweiligen Ordner beschreibt kurz den gewünschten Clip.
- Es gibt keine Trennung mehr in `desktop` und `mobil`.
- Bestehende echte Videos können ebenfalls direkt im jeweiligen Bereichsordner liegen.
