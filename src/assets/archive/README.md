This folder keeps image files that are not currently imported by the website.

`source-images/`
Original full-size photos used to generate the responsive runtime variants in `src/assets/images/`.

`reference-photo-library/`
Additional project/reference photos kept for future content, but not shipped in the current app.
These files use category-first names such as `ceiling-construction-reference-01.jpg` for easier browsing.

The production image set should stay in `src/assets/images/` and only contain files that are actively used by the frontend.
